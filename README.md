# forms-creator
Javascript library using jQuery selecters to create and format input forms on the fly!<br>
This include input formats for:
  <ul>
    <li>Social security number</li>
    <li>Money</li>
    <li>Names</li>
    <li>Date</li>
    <li>Phone</li>
    <li>Credit Card</li>
    <li>Selection options</li>
  </ul>
